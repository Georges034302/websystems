Web Systems Demo Project

An educational project demonstrating core web technologies: HTML structure, CSS styling, and JavaScript behaviour.

Project Structure

websystems/
├── index.html         — Home page with live clock
├── html/
│   ├── navbar.html    — Shared navigation bar (loaded dynamically)
│   ├── footer.html    — Shared footer (loaded dynamically)
│   ├── tools.html     — CS tools page (factorial calculator)
│   └── about.html     — This page
├── css/
│   ├── base.css       — Global reset and shared styles
│   ├── navbar.css     — Navigation bar styles
│   ├── footer.css     — Footer styles
│   ├── index.css      — Home page clock styles
│   ├── tools.css      — Tools page styles
│   └── about.css      — About page styles
└── js/
    ├── layout.js      — Loads navbar and footer into all pages
    ├── index.js       — Live clock using Date and setTimeout
    └── tools.js       — Factorial function and calculator logic
